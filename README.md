# template-flask

## Installation

```bash
pip install requirements.txt
```
<details>
    <summary>Additional Information</summary>

    pip install flask
</details>
<br>

### Start Project

1\. Run Server  
```bash
flask run
```
<details>
    <summary>Additional Information</summary>

- bash
    ```bash
    $ export FLASK_APP=hello
    $ flask run
    * Running on http://127.0.0.1:5000/
    ```

- powershell    
    ```powershell
    > $env:FLASK_APP = "hello"
    > flask run
    * Running on http://127.0.0.1:5000/
    ```
 </details>
